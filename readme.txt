== Description ==

This plugin places a contact form that is geared towards freelance developers/designers on any Page or Post via shortcode. The form fields are: First Name (text), Last Name (text), Subject (text), Category (select box), Budget (select box), Email (text), Phone (text), Website (text), Message (textarea).

Each form submission is submitted as a Custom Post Type called "Contact". You can view the form submissions from your WordPress admin under "Contact". This allows for easier sorting and storing of Contact form submissions.

*Note, this plugin requires [Easy CPT](http://zanematthew.com/blog/plugins/easy-custom-post-types/)*

=== Features ===
* AJAX form submission
* Automatically backed up to your WordPress database
* Multiple ways to sort form field submissions

== Installation ==

1. Install the plugin via WordPress or download and upload the plugin to the `/wp-content/plugins/`
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add the following shortcode to any Post or Page `freelance_contact`
